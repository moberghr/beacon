# Internal Working Documents

This directory holds engineering working artifacts — implementation plans, design
notes, historical audits, and archived specs. Nothing here is part of the public
product documentation; see the [README](../README.md) and the
[documentation site](https://mibu.github.io/semantico) for that.

| Path | What it is |
|---|---|
| `ClaudePlans/` | Historical implementation plans and diffs (React migration, query chaining) |
| `Specify/` | Spec-kit scaffolding used during early feature planning |
| `specs/` | Early feature specs and roadmap brainstorms |
| `docs-archive/` | Superseded docs drafts (multi-agent documentation experiments, changelogs) |
| `UNFINISHED.md` | Tracker for features hidden from the UI pending redesign |
| `MULTI_AGENT_*.md`, `INTEGRATION_COMPLETE.md` | Multi-agent documentation system working notes |
| `CODEBASE_CLEANUP_FINDINGS.md` | Historical cleanup audit |
| `CLAUDE_STYLE_GUIDE.md` | Superseded by `.claude/references/dotnet/coding-guidelines.md` |
